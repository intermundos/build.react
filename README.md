# React skeleton

## Description

>React skeleton project. Includes redux, router, css framework bulma.io, antd components library.

## Inside

- React
- Redux
- Redux Saga
- React Router v4
- webpack 2
- SASS
- postcss
- rucksack-css
- bulma.io
- antd



## Instructions:

1. yarn install 
2. yarn run dev
3. open localhost:8888
